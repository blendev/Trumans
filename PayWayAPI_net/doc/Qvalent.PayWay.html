Class Description
PayWayAPI This class is the client for the PayWay Credit Card API. The intended use of this class is that you create one CCAPIClient instance in your application, then call it from multiple threads to process credit card transactions. This class is thread-safe, so using one instance from many threads is safe. You must call initialise() on this object before it can be used.
